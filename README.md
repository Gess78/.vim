# .vim
## Vim configuration


My `.vimrc` configuration file.


Installation
------------

### Linux

    $ cd
    $ rm -rf .vim
    $ git clone https://github.com/Gess78/.vim.git
    $ ln -s .vim/.vimrc
    
    # or
    cd&&\
    rm -rf .vim&&\
    git clone https://github.com/Gess78/.vim.git &&\
    ln -s .vim/.vimrc
    
To install plugins run in vim: 

    PlugInstall
